# NVivo (by Lumivero) - Competitive Profile

**Last Updated**: 2026-03-03
**Confidence**: High (multiple verified sources)
**Threat Level**: Medium

---

## Overview

- **Product**: NVivo (qualitative data analysis software)
- **Company**: Lumivero LLC (formerly QSR International, rebranded 2022)
- **Founded**: NVivo lineage traces back to NUD*IST (1981); NVivo brand since 1999
- **Ownership**: TA Associates (private equity) — formed Lumivero in 2022 by combining QSR International + Palisade + Addinsoft
- **Key Acquisitions**: ATLAS.ti (Sept 2024), Risk Decisions Ltd (2024)
- **Target Market**: Academic researchers (social sciences, education, health, public policy), government evaluators, market researchers
- **Positioning**: "The #1 qualitative data analysis software for 30 years" — most cited QDA in publications (Scopus, 2010-2023)
- **Platforms**: Windows (full), Mac (near-parity as of v15.3), no web/browser version
- **Current Version**: NVivo 15.3 (December 2025)
- **Market Context**: QDAS market valued at USD 1.2B (2024), projected USD 1.9B by 2032

## Features

### Tablestakes
- Manual coding, code hierarchies, memos, annotations
- Text search and word frequency queries
- Import documents, PDFs, audio, video, images
- Code-and-retrieve, data export, project organization

### Differentiators
- **Framework Matrix**: Case-by-theme analysis with AI-generated cell summaries (v15.3)
- **Academic Citation Dominance**: Most-cited QDA tool; deep institutional adoption
- **Statistical Integration**: Crosstabs, PCA biplot, box plots — stronger quant-qual bridge
- **Methodology Templates**: Thematic Analysis, Grounded Theory, Phenomenology, Ethnography, Discourse, Narrative, Content Analysis, Case Studies
- **Transcription**: Built-in, 40+ languages, ~90% accuracy
- **Autocoding Variants**: By pattern, word frequency, headings, speakers

### Emerging
- AI Summaries for codes, cases, Framework Matrix cells (v15.3, Dec 2025)
- AI-Suggested Child Codes
- Admin AI Controls (institutional kill switch for IRB compliance)
- Mac Feature Parity push (Framework Matrix on Mac)
- PCA Visualization (Windows)
- SaaS direction signaled by CEO post-ATLAS.ti acquisition

### Deprecated
- NVivo Starter Edition (discontinued)
- True cross-platform parity (still gaps)
- Browser/Web version (never launched)
- Mobile coding (no app)

## User Sentiment

**G2**: ~4.1/5 | **Capterra**: ~4.2/5

**Loves**: Comprehensive coding/query system, word clouds/hierarchy charts, Framework Matrix, academic credibility
**Complaints**: Steep learning curve, sluggish with large projects (70+ sources), Collaboration Cloud "catastrophic", poor customer support (months-long responses), expensive TCO, mediocre PDF handling, dated UI

## Strengths
1. Institutional lock-in (university curricula, grant workflows)
2. Citation dominance (self-reinforcing academic cycle)
3. Comprehensive methodology coverage (8+ traditions)
4. Lumivero portfolio synergy (NVivo + ATLAS.ti + XLSTAT)
5. Framework Matrix (no equivalent elsewhere)
6. Built-in transcription (40+ languages)

## Weaknesses
1. Desktop-only (no web in cloud-native era)
2. Collaboration Cloud unreliable (sync failures, data corruption)
3. Performance degrades at scale (70+ sources, 100+ codes)
4. Poor customer support
5. Expensive total cost of ownership
6. No mobile app
7. Conservative AI (summaries only)
8. **PDF handling described as "mediocre"** — direct gap for Qualia
9. Dated ribbon UI
10. PE ownership risk (extraction > innovation)

## Key Takeaways for Qualia Coding

1. PDF/image coding is a real gap — NVivo's PDF is "mediocre" with no bookmarks
2. Performance at scale is an opportunity — NVivo chokes on 70+ sources
3. Lumivero monopoly may push users to seek independent alternatives
4. Modern Obsidian-native architecture vs dated desktop approach
5. AI is table stakes but NVivo's implementation is weak
