# ATLAS.ti (Lumivero) - Competitive Profile

**Last Updated**: 2026-03-03
**Confidence**: High (multiple corroborating sources)
**Threat Level**: Low-Medium

---

## Overview

- **Product**: ATLAS.ti — qualitative data analysis software
- **Company**: ATLAS.ti Scientific Software Development GmbH (acquired by Lumivero Sept 2024)
- **Founded**: 1993 (first commercial release by Thomas Muhr)
- **Headquarters**: Berlin, Germany
- **Employees**: ~50-200
- **Parent**: Lumivero (also owns NVivo, Citavi, Risk Decisions)
- **Target Market**: Academic researchers, social scientists, UX researchers
- **Positioning**: "The #1 Software for Qualitative Data Analysis" — AI-powered qualitative insights
- **Platforms**: Windows, Mac, Web, Mobile (cross-platform under single license)

## Pricing

| Tier | Price | Notes |
|------|-------|-------|
| Commercial Perpetual | $670 | Windows or Mac |
| Academic | ~$110/year | Requires institutional affiliation |

AI features are **token-metered** — not unlimited. Trial advertised as "no limitations" but capped at 500-word documents.

## Features

### Tablestakes
- Manual coding, code hierarchies, memos, annotations
- PDF support, search/retrieval, reports, multi-format import
- Project bundling and sharing

### Differentiators
- **Margin Area with Hyperlinks**: Unique UX showing codes, links, memos in margin
- **Quotation-Centric Data Model**: Quotation (not code) is foundational unit — encourages deeper data engagement
- **Network Views**: Best-in-class concept mapping among traditional QDAS
- **Video Coding with Synchronized Transcripts**: Fine-tuned, superior to NVivo/MAXQDA
- **Grounded Theory Workflow**: Purpose-built; memo/quotation/network tools form coherent methodology
- **Cross-Platform**: Win + Mac + Web + Mobile under one license
- **Co-occurrence → Code Conversion**: Can convert overlapping themes into codes (v25)

### Emerging
- AI Transcription (30+ languages, speaker detection) — Dec 2024
- Conversational AI (chat with documents via OpenAI)
- AI Coding with granularity slider (broad → specific)
- AI Summaries (had stability issues, fixed v25.0.2)
- Sentiment Analysis Coding (enhanced Jan 2026)
- EU/US Data Residency for AI processing

### Deprecated
- Standalone mobile apps (focus shifting to web)
- SPSS/stats integration (rarely mentioned now)
- On-premise team server (replaced by cloud)

## User Sentiment

**G2**: ~4.5/5 | **Capterra**: ~4.0-4.5/5

**Loves**: Coding workflow, data type versatility, visualizations (network views, word clouds), AI coding suggestions, web collaboration, customizable workspace
**Complaints**: **Data loss (~80% quotations disappearing after updates)**, 10-15s per code application at scale, misleading trial, "dreadful" customer support, AI token limits, no hotkey customization

## Strengths
1. 30+ year brand recognition and academic entrenchment
2. Unique quotation-centric data model
3. Multi-data-type support (text, PDF, image, audio, video, social media, surveys)
4. Widest AI feature set in QDAS
5. Cross-platform (Desktop + Web + Mobile)
6. Best-in-class network visualization
7. Lumivero backing (shared R&D)

## Weaknesses
1. **Performance degradation at scale** — fundamental architecture issue
2. **Data integrity concerns** — data loss reports devastating for research
3. **Token-metered AI** — artificial scarcity, unpredictable costs
4. **Expensive for individuals** ($670 perpetual)
5. **Consolidation uncertainty** — Lumivero owns NVivo+ATLAS.ti, product rationalization risk
6. **OpenAI dependency** — privacy concerns for IRB-regulated research
7. **Misleading trial** — erodes trust
8. **Closed ecosystem** — no plugins, no API, no integrations

## Key Takeaways for Qualia Coding

1. Lumivero monopoly creates opportunity for independent alternatives
2. AI is table stakes — but ATLAS.ti's cloud-dependent, token-metered approach has real weaknesses a local-first approach could exploit
3. Data loss is ATLAS.ti's Achilles' heel — Qualia's file-based storage is inherently more resilient
4. Zero PKM integration — no Obsidian/Notion/Roam connection
5. No CSV/Parquet support — Qualia fills this gap
6. Cloud-first push alienates privacy-conscious researchers — local-first is a differentiator
