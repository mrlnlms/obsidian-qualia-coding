# MAXQDA (by VERBI Software) - Competitive Profile

**Last Updated**: 2026-03-03
**Confidence**: High (official website, G2, Capterra, release notes, job postings)
**Threat Level**: HIGH — most direct competitor

---

## Overview

- **Product**: MAXQDA — qualitative and mixed methods data analysis software
- **Company**: VERBI Software GmbH (family-owned, privately held)
- **Founded**: 1989 (as "MAX" for DOS; MAXQDA brand launched 2001)
- **Headquarters**: Berlin, Germany
- **Leadership**: Isabel Kuckartz (CEO), Anne Kuckartz (CEO) — second-generation family
- **Employees**: ~62
- **Revenue**: ~$23M estimated
- **Funding**: None — bootstrapped, family-owned
- **Global Reach**: 150+ countries
- **Positioning**: "The #1 qualitative data analysis software with the best AI integration"
- **Notable Clients**: Amazon, Plan International, University of Oxford, World Vision

## Pricing

Subscription-only (no perpetual licenses).

| Tier | Academic (annual) | Business (3-yr) | Non-Profit (3-yr) |

**Add-Ons**: AI Assist Premium (€120/yr), Transcription (€49-78), TeamCloud (annual)

## Features

### Tablestakes
- Text coding, code hierarchies, memos, text search, variables
- Multi-format import (DOCX, RTF, TXT, PDF), audio/video transcription
- Image coding (basic), intercoder reliability (Kappa)
- REFI-QDA import/export

### Differentiators
- **Mixed Methods Gold Standard**: Joint Displays, Crosstabs, Typology Tables, QTT Worksheets (Creswell)
- **MAXDictio**: Built-in quantitative text analysis (dictionary-based, lemmatization 13+ languages, KWIC, word trees)
- **20+ Interactive Visualizations**: MAXMaps, code portraits, document portraits, co-occurrence matrices, clustering — all linked to source data
- **Analytics Pro Statistics**: Chi-square, correlation, t-tests, ANOVA, Mann-Whitney U, Cronbach's Alpha, cluster analysis — no SPSS needed
- **Cross-Platform Parity**: Identical Win/Mac since 2015
- **Gentle Learning Curve**: Consistently cited as easiest traditional QDAS
- **Excellent Support**: 24-hour response, "first rate" reviews

### Emerging
- AI Assist (2023): summaries, paraphrasing, subcodes, "chat with documents"
- Multi-document AI Coding (Feb 2026)
- AI Translation (Feb 2026)
- MAXQDA Tailwind (cloud AI platform)
- TeamCloud collaboration
- Continuous subscription model (dropped versioned releases)
- Survey Analysis Workspace (26.0)

### Deprecated
- Perpetual licenses (eliminated)
- Version-based releases
- Linux support (never offered)
- Web/browser version (not available)

## User Sentiment

**Capterra**: 4.7/5 | **G2**: 4.5+/5

**Loves**: Intuitive interface, powerful mixed methods, comprehensive visualizations, lightweight, excellent support, cross-platform, MAXDictio
**Complaints**: Price barrier, collaboration "precarious", no Linux, no web version, restrictive licensing, video coding limitations, subscription friction

## Strengths
1. Mixed methods gold standard — best qual+quant integration
2. 35-year academic pedigree, Creswell framework integration
3. True Mac/Win parity
4. 20+ visualization types linked to source data
5. Gentlest learning curve
6. Built-in statistics (no SPSS needed)
7. Excellent support quality
8. Community ecosystem (MAXDAYS, Academy, trainer network)

## Weaknesses
1. **Desktop-only** — no web, no mobile
2. **Collaboration is bolted on** — TeamCloud is paid add-on, limited
3. **AI as upsell** — €120/yr add-on; competitors bundling AI in base
4. **Subscription backlash** — academics used to perpetual licenses
5. Small team (~62) — limited engineering bandwidth
6. No Linux
7. **Video coding transcript-centric** — inferior to ATLAS.ti and Qualia's WaveSurfer
8. Desktop architecture limits iteration speed

## Key Takeaways for Qualia Coding

1. Mixed methods is table stakes — MAXQDA set the bar
2. Collaboration is the gap (shared with Qualia currently)
3. AI bundling wins — including AI in base product differentiates
4. **Visual media coding is underserved** — Qualia's Fabric.js and WaveSurfer engines offer superior experience
5. **Obsidian angle is the biggest differentiator** — MAXQDA is a standalone silo; Qualia inherits the entire PKM ecosystem
6. Data format breadth — Qualia adds CSV, Parquet, Analytics beyond MAXQDA's scope
7. Price sensitivity — MAXQDA's €253+/yr leaves room for a free alternative
