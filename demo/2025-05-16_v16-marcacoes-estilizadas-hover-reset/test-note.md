# v16 — Marcacoes estilizadas, hover OK, reset codigos

## What's new
- Marcacoes com estilo visual (background colorido, border-radius)
- Hover effect funcional nas marcacoes (brightness filter + classe JS)
- Comando para resetar todas as marcacoes salvas
- Handles com bolinhas estilizadas e barrinhas verticais
- CSS refinado: cursores distintos nas alcas (w-resize / e-resize)

## How to verify
1. Open console: should show `[CodeMarker] v16 loaded`
2. Select text and run command "Criar uma nova marcacao de codigo"
3. Hover over the highlighted text — should see brightness effect
4. Run command "Resetar todas as marcacoes salvas" — should clear all markers
5. Check that handles appear with styled balls and vertical bars
