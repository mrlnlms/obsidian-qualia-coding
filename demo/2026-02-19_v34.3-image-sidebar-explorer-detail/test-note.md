# v34.3 — Sidebar Views: Code Explorer + Code Detail

## What's new
- **Code Explorer view**: tree-style sidebar panel showing all coded image regions organized by code/tag
- **Code Detail view**: sidebar panel showing details of a selected code marker (label, comment, coordinates)
- Two new registered view types for the sidebar

## How to verify
1. Open an image file via right-click > "Open in CodeMarker Image"
2. Add some code markers to regions on the image
3. Check the right sidebar for "Code Explorer" and "Code Detail" panels
4. Click on entries in the Explorer to see details update in the Detail panel
5. Verify the tree structure groups markers by their assigned codes
