# What's new in v36.3

- **Timeline ruler** — visual time markers along the waveform for precise navigation
- **Region lanes** — coded regions displayed as colored lanes below the waveform
- **registerExtensions** — audio file extensions now properly registered with Obsidian
- **Scroll persistence** — timeline scroll position preserved across interactions

# How to verify

1. Open an audio file (e.g., `Conquerors and The World.mp3`) in the audio view
2. Check that a timeline ruler with time markers appears along the waveform
3. Create some coded regions and verify they appear as colored lanes
4. Scroll the timeline, interact with controls, and confirm scroll position is preserved
5. Check console for `[obsidian-codemarker-audio] v36.3 loaded`
