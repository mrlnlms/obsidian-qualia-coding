# What's new in v34.2

- **Coding menu**: Right-click on image regions to assign qualitative codes
- **Region labels**: Coded regions now display their assigned code name
- **Hover glow**: Hovering over a region highlights it with a glow effect

# How to verify

1. Open an image file (png/jpg) — it should open in the CodeMarker Image view
2. Draw a region on the image using the rectangle or ellipse tool
3. Right-click on the drawn region — a coding menu should appear
4. Assign a code to the region — a label should appear on/near the region
5. Hover over a coded region — it should glow to indicate selection
