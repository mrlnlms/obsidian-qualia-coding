# v38.3 — Dashboard KPIs + Thumbnails

## What's new
- Dashboard view as landing page for Analytics
- KPI cards showing summary statistics (total codes, annotations, documents)
- Mini-thumbnails previewing each visualization (Network Graph, Matrix, Code Evolution)

## How to verify
1. Open the Analytics sidebar (codemarker-analytics)
2. Confirm the Dashboard loads as the default landing view
3. Check that KPI numbers are displayed at the top
4. Verify mini-thumbnails are visible for each available view mode
5. Click a thumbnail to navigate to that visualization
