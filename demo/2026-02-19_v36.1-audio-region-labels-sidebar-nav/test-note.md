# CodeMarker Audio v36.1 — Region Labels + Sidebar Nav

First appearance of the Audio engine in the demo vault.

## What's new in v36.1

- **Audio engine scaffold**: waveform visualization for audio files (mp3, m4a, wav, ogg, flac, aac)
- **Region labels**: clickable chips on the waveform showing coded regions
- **Sidebar navigation**: Code Explorer and Code Detail views for audio codes
- **Playback controls**: play/pause, zoom, volume, speed adjustment
- **Coding model**: qualitative coding on audio regions with time-based markers

## How to verify

1. Open the console — look for `[CodeMarker Audio] v36.1 loaded`
2. Open `Conquerors and The World.mp3` from the vault — it should open in the CodeMarker Audio view with a waveform
3. Try the playback controls (play, zoom, speed)
4. Right-click on the waveform to create a coded region
5. Check sidebar views: Code Explorer and Code Detail should be available
